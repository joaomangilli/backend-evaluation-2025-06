# Mock Payment API

A simple Sinatra application used for testing payment interactions.

## Setup

```
bundle install
```

## Running the app

```
bin/dev
```
The service listens on port `4000` by default.

Swagger documentation is available in a UI at `http://localhost:4000/api-docs`.

## Running tests

```
bin/test
```

## Linting

```
bin/rubocop
```
